# **App Name**: CollabZoom

## Core Features:

- Room Creation: Generate a unique code for a new video conference room.
- Room Joining: Allow users to join a video conference room using a code.
- Video/Audio Streaming: Enable real-time video and audio streaming using WebRTC.
- Screen Sharing: Allow participants to share their screen with others in the meeting.
- In-Call Messaging: Provide a text-based chat feature for participants to communicate during the call.
- Room Code Management: Store and manage the active room codes and related metadata in Firestore.

## Style Guidelines:

- Primary color: Sky blue (#74B9FF), a calming and trustworthy hue.
- Background color: Very light blue (#E6F3FF), creating a clean and unobtrusive backdrop.
- Accent color: Light purple (#B19CD9), adding a touch of sophistication without being distracting.
- Body and headline font: 'Inter', a sans-serif font providing a modern and readable feel. Note: currently only Google Fonts are supported.
- Use clean, minimalist icons for call controls (mute, share screen, leave, etc.).
- Simple, intuitive layout with prominent video feeds and easy-to-access controls.
- Subtle transitions when users join/leave the call or share their screen.
